❓ Como Usar o Gerenciador de Mangás

📚 Aba Mangás

Visualize todos os seus mangás cadastrados. Use o botão "Adicionar Mangá" para incluir novos títulos manualmente.

🌐 Aba Sites

Gerencie os sites de leitura suportados. Adicione novos sites para ter controle de onde ler suas obras favoritas. E ele detecatar automaticamente que você está em um mangá.

📖 Aba Histórico

Acompanhe seu histórico de leitura com filtros por período.

⚙️ Aba Configurações

Configure a detecção automática, selecione arquivo de dados e faça backup dos seus dados.

💾 Exportar/Importar

Use os botões na parte superior para fazer backup dos seus dados em formato JSON.

💰 Doações

Se você gosta da extensão, considere fazer uma doação simbólica via PayPal para apoiar o desenvolvimento.
